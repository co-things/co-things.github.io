hydra教程

发表于 2015-09-09   |  

hydra是一款用于暴力破解的工具,支持HTTP/HTTPS/Mysql/ssh等协议,至于有什么用,你懂的…

下载

wget https://www.thc.org/download.php?t=r&f=hydra-8.1.tar.gz

用法

hydra
       [[[-l LOGIN|-L FILE] [-p PASS|-P FILE|-x OPT]] | [-C FILE]] [-e nsr]
       [-u] [-f] [-F] [-M FILE] [-o FILE] [-t TASKS] [-w TIME] [-W TIME]
       [-s PORT] [-S] [-4/6] [-vV] [-d]
       server service [OPTIONAL_SERVICE_PARAMETER]


-R
继续从上一次进度接着破解
-S
大写，采用SSL链接
-s 
小写，可通过这个参数指定非默认端口
-l 
指定破解的用户，对特定用户破解
-L 
指定用户名字典
-p 
小写，指定密码破解，少用，一般是采用密码字典
-P 
大写，指定密码字典
-e 
可选选项，n：空密码试探，s：使用指定用户和密码试探
-C < FILE>
使用冒号分割格式，例如“登录名:密码”来代替-L/-P参数
-M < FILE>
指定目标列表文件一行一条
-o < FILE>
指定结果输出文件
-f
在使用-M参数以后，找到第一对登录名或者密码的时候中止破解
-t < TASKS>
同时运行的线程数，默认为16
-w < TIME>
设置最大超时的时间，单位秒，默认是30s
-v / -V
显示详细过程
server
目标ip
service
指定服务名，支持的服务和协议：telnet ftp pop3[-ntlm] imap[-ntlm] smb smbnt http[s]-{head|get} http-{get|post}-form http-proxy cisco cisco-enable vnc ldap2 ldap3 mssql mysql oracle-listener postgres nntp socks5 rexec rlogin pcnfs snmp rsh cvs svn icq sapr3 ssh2 smtp-auth[-ntlm] pcanywhere teamspeak sip vmauthd firebird ncp afp等等
OPT
可选项

实例

破解http登录(GET/POST)

hydra -L 用户名 -P 密码字典 -t 线程 -s 端口 ip http-[get|post]-form '/path:param1=参数1:account=^USER^&password=^PASS^:错误提示:H=Cookies\: a=b'

^USER^表示账号 ^PASS^表示密码
PS:要注意对参数里面的冒号转义

破解redis

hydra -P 密码字典 -s 端口 ip redis

HTTP登录破解测试

创建页面test.php,当输入用户名为admin,密码为asdasd时,页面输出success,否则为error

创建账号密码字典account.txt,password.txt

# cat account.txt
john
admin
root
# cat password.txt
123
aaaaaa
123456
password
asdasd
test

指定字典，开始暴力破解

# hydra -L account.txt -P password.txt -s 8080 127.0.0.1 http-post-form '/test.php:account=^USER^&password=^PASS^:error' -t 4
